# Smart Travel Tracker API 🚀


This API enhances travel-related messages by:
✅ Detecting locations 📍  
✅ Converting time zones 🕒  
✅ Converting currencies 💱  



### **Modify Message**
**POST** `/modify-message`  
Modifies a travel-related message.

#### 📌 Request:
```json
{
  "message": "Convert 100 USD to EUR"
}
